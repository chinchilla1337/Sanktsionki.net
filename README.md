## Идея компании
После ухода многих международных брендов одежды (Zara, H&M, Nike, Adidas, Levi's и др.) из России, покупка оригинальных вещей превратилась в квест с переплатами и рисками подделок. **Sanktsionki.net** — это аналитическая платформа для поиска лучших предложений для покупки ушедших брендов на международном рынке одежды. 
Платформа автоматически рассчитывает финальную стоимость вещи с учетом комиссии байеров, курса валюты страны покупки, стоимости логистики и пошлины.

## Механизм работы нашей платформы
*   Анализ данных российских платформ для покупки брендовой одежды (Ozon, Wildberries, Avito, Дэву, Lamoda, CDEK.Shopping, Яндекс Маркет) и зарубежных платформ (официальные сайты брендов).
*   Отслеживание цен в реальном времени и сравнение с российским офлайн-рынком.
*   Учет данных о комиссиях популярных сервисов доставки.

## Разделение обязанностей
1.  **Анастасия Зубко** — анализ российского рынка брендовой одежды; построение графиков наценки; аналитический дашборд (таблица, где пользователь вводит артикул товара, а система выдает самую дешевую платформу для покупки).

2.  **Вероника Шалимова** — анализ зарубежного предложения; создание API через Google Sheets, куда стекаются данные от всех участников; анализ чатов байеров для выявления среднего чека на услуги посредников.
    
    
## Как мы будем собирать данные?
*    Парсинг основных параметров одежды (цены, артикулы, модели, цвет, бренд и т. д.) на зарубежных и российских сайтах через API, Selenium, requests.
*    Анализ стоимости услуг байеров (логгирование чатов) через парсинг мессенджеров.
*    Поиск магазинов, которые физически остались, но работают как стоки, через парсинг 2ГИС или Яндекс Карт.

## Целевая аудитория
*   Люди, которые хотят купить оригинальный товар без 200% наценки перекупа.
*   Байеры и реселлеры для мониторинга конкуренции и поиска лучших цен закупки за рубежом.
*   Те, кто не хочет нарваться на подделку на маркетплейсах.
