## Идея компании
После ухода многих международных брендов одежды (Zara, H&M, Nike, Adidas, Levi's и др.) из России, покупка оригинальных вещей превратилась в квест с переплатами и рисками подделок. **Sanktsionki.net** — это аналитическая платформа для поиска лучших предложений для покупки ушедших брендов на международном рынке одежды. 
Платформа автоматически рассчитывает **финальную стоимость вещи** с учетом комиссии байеров, курса валюты страны покупки, стоимости логистики и пошлины.

## Как это работает?
*   Анализ данных **российских платформ для покупки брендовой одежды** (Ozon, Wildberries, Avito, Дэву, Lamoda, CDEK.Shopping, Яндекс Маркет) и **зарубежных платформ** (официальные сайты брендов).
*   Отслеживание цен в реальном времени и сравнение с российским офлайн-рынком.
*   Учет данных о комиссиях популярных сервисов доставки.

## Разделение обязанностей
1.  **[Анастасия Зубко]** — анализ российского рынка брендовой одежды**.
    *   Парсинг: Avito, The Cultt, Resale площадки Ozon/WB.
    *   Визуализация: Построение графиков наценки на ресейле (разница между ценой в Дубае и ценой в Москве у перекупа).
2.  **[Вероника Шалимова]** — **Анализ зарубежного предложения (Турция/ОАЭ/СНГ)**.
    *   Парсинг: Trendyol (Турция), Namshi (ОАЭ), официальные сайты Inditex (Zara Home) в Казахстане.
    *   Геоаналитика: Карта офлайн-магазинов ушедших брендов в Москве и стока (Садовод, Москва-Сити).
    *   NLP/Парсинг отзывов: Анализ чатов байеров в Telegram (через Telethon) для выявления среднего чека на услуги посредников.
3.  **[Имя 3, Тимлид]** — **Архитектура и расчет цены "под ключ"**.
    *   Парсинг: Poizon (Китай) для кроссовок и стритвира.
    *   Backend: Создание API через Google Sheets, куда стекаются данные от всех участников.
    *   Аналитический дашборд: Таблица, где пользователь вводит артикул товара, а система выдает **самую дешевую страну для выкупа с учетом всех комиссий**.

## Как собирать данные?
*   **API / Selenium**: Обход блокировок зарубежных сайтов и сбор артикулов.
*   **Requests**: Прямые запросы к скрытым API маркетплейсов.
*   **Парсинг Telegram**: Анализ стоимости услуг байеров (логгирование чатов).
*   **Парсинг 2ГИС / Яндекс Карт**: Поиск магазинов, которые физически остались, но работают как стоки.

## Кому это нужно?
*   Людям, которые хотят купить **оригинальный** Massimo Dutti или Uniqlo без 200% наценки перекупа.
*   Самим **байерам и реселлерам** для мониторинга конкуренции и поиска лучших цен закупки за рубежом.
*   Тем, кто не хочет нарваться на подделку на маркетплейсах.
